# Руководство по просмотру комиксов

## Обзор

В приложении Mahabharata теперь реализован новый просмотрщик комиксов с вертикальным скроллом, аналогичный legacy iOS версии. Этот просмотрщик обеспечивает плавный опыт чтения с поддержкой прикосновений и оптимизированной загрузкой контента.

## Основные возможности

### 1. Вертикальный скролл
- **Непрерывная прокрутка**: В отличие от постраничного просмотра, новый просмотрщик использует вертикальный скролл, как в legacy версии
- **Прикосновения**: Управление через прикосновения для естественного взаимодействия
- **Плавность**: Плавная анимация при прокрутке

### 2. Ленивая загрузка
- **Оптимизация памяти**: Слои загружаются только когда они находятся в видимой области или рядом с ней
- **3x область**: Загружаются слои в области 3x высоты экрана для обеспечения плавности
- **Автоматическая выгрузка**: Слои вне видимой области автоматически выгружаются

### 3. Адаптивный масштаб
- **Подгонка по ширине**: Комикс автоматически масштабируется под ширину экрана
- **Настраиваемый zoom**: Пользователь может изменять масштаб от 50% до 200%
- **Сохранение пропорций**: Соотношение сторон сохраняется при масштабировании

### 4. Интерфейс управления
- **Скрытие контролов**: Тап по экрану скрывает/показывает элементы управления
- **Прогресс бар**: Отображает текущий прогресс чтения
- **Настройки**: Доступ к настройкам масштаба и информации о комиксе

## Архитектура

### Компоненты

```
lib/features/episode/
├── episode_screen.dart              # Экран эпизода
└── widgets/
    ├── comics_viewer.dart           # Старый постраничный просмотр (deprecated)
    ├── comics_scroll_viewer.dart    # Новый скролл-просмотр ⭐
    └── comics_viewer_new.dart       # FreeDome плеер (опционально)
```

### ComicsScrollViewer

Главный виджет для просмотра комиксов со скроллом.

**Основные возможности:**
- `ScrollController` для управления прокруткой
- Ленивая загрузка слоев на основе видимости
- Адаптивный масштаб под размер экрана
- Элементы управления с автоскрытием

**Параметры:**
```dart
ComicsScrollViewer({
  required String comicsFilePath,  // Путь к файлу .comics
  required String episodeTitle,     // Название эпизода
})
```

### ComicsService

Сервис для работы с файлами комиксов (архивы ZIP).

**Методы:**
- `getComicsMetadata()` - Получение метаданных комикса
- `getComicsLayers()` - Получение списка слоев
- `getLayerImage()` - Получение изображения конкретного слоя
- `getPageImage()` - Получение изображения страницы (для старого просмотра)

## Формат данных

### Структура .comics файла

Файл .comics является ZIP-архивом со следующей структурой:

```
mahabharata_episode.comics
├── manifest.json или metadata.json  # Метаданные
├── layers.json или data.json        # Информация о слоях
└── layers/                          # Папка с изображениями слоев
    ├── layer_001.png
    ├── layer_002.png
    └── ...
```

### Формат layers.json

```json
{
  "width": 1080,
  "height": 15000,
  "layers": [
    {
      "index": 0,
      "imagePath": "layers/layer_001.png",
      "y": 0,
      "height": 800,
      "width": 1080,
      "alpha": 1.0
    },
    ...
  ]
}
```

### Формат metadata.json

```json
{
  "title": "Episode 1 - Book 1",
  "author": "Igor Baranko & Alexey Chebykin",
  "description": "First chapter of Mahabharata",
  "duration": 300,
  "audioFile": "audio/episode1.mp3",
  "width": 1080,
  "height": 15000
}
```

## Оптимизация производительности

### Ленивая загрузка

Слои загружаются только когда они видимы или близки к видимой области:

```dart
// Область видимости с запасом
final visibleTop = scrollOffset - viewportHeight;
final visibleBottom = scrollOffset + viewportHeight * 2;

// Проверка пересечения
if (layerBottom >= visibleTop && layerTop <= visibleBottom) {
  _loadedLayers.add(layer.index);
}
```

### Управление памятью

- Автоматическая выгрузка слоев вне видимой области
- Использование `FutureBuilder` для асинхронной загрузки
- Кэширование загруженных изображений

### Производительность скролла

- `ClampingScrollPhysics` для точного контроля
- Оптимизированный рендеринг с помощью `Positioned`
- Минимальные перестроения виджетов

## Известные проблемы

### Текущие ограничения

1. Нет поддержки звуковых анимаций (в разработке)
2. Нет матричных трансформаций слоев (планируется)
3. Демо-слои используются если файл не найден

### Решения

Для тестирования без реальных файлов используются демо-слои:

```dart
List<ComicsLayer> _createDemoLayers() {
  final demoLayers = <ComicsLayer>[];
  for (int i = 0; i < 20; i++) {
    demoLayers.add(ComicsLayer(
      index: i,
      imagePath: 'assets/images/layer_$i.png',
      y: i * 800.0,
      height: 800.0,
      width: 1080.0,
      alpha: 1.0,
    ));
  }
  return demoLayers;
}
```
